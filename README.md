# 💄 Claudia Vera - Makeup & Hair

Sitio web profesional para Claudia Vera, maquilladora y peinadora especializada en maquillaje social y alto impacto con más de 12 años de experiencia.

## 🌟 Características

### Diseño Responsivo
- Interfaz moderna y elegante optimizada para dispositivos móviles, tablets y desktop
- Paleta de colores sofisticada con tonos rosados y grises neutros
- Animaciones suaves que mejoran la experiencia del usuario

### Secciones Principales
- **Hero**: Presentación inicial atractiva con call-to-action
- **Sobre mí**: Información personal y experiencia profesional
- **Servicios**: 6 servicios principales con detalles interactivos
  - Maquillaje Social
  - Peinado Social
  - Alto Impacto
  - Perfilado de Cejas
  - Talleres
  - Pelito Express
- **Trabajos**: Galería de proyectos completados
- **Opiniones**: Testimonios de clientes satisfechas
- **Footer**: Información de contacto y créditos

### Funcionalidades
- **Modal Interactivo**: Detalles de servicios con integración directa a WhatsApp
- **Scroll Optimizado**: Navbar con efecto glass-morphism al scroll
- **Animación de Reveal**: Elementos que aparecen con efecto al desplazarse
- **Loader Personalizado**: Pantalla de carga con animaciones
- **Integración WhatsApp**: Botón flotante y CTA directos al chat
- **Año Dinámico**: Footer actualizado automáticamente

## 🛠️ Tecnologías Utilizadas

- **HTML5**: Estructura semántica
- **CSS3**: Estilos personalizados con animaciones
- **JavaScript**: Interactividad y lógica del cliente
- **Tailwind CSS**: Framework CSS de utilidades para diseño rápido
- **Lucide Icons**: Iconografía moderna y limpia
- **Google Fonts**: Tipografía Poppins

## 🚀 Cómo Usar

### Visualizar el Proyecto
1. Clona el repositorio:
   ```bash
   git clone https://github.com/NunezHoracio/claudiaveramakeup.git
   ```

2. Navega a la carpeta del proyecto:
   ```bash
   cd claudiaveramakeup
   ```

3. Abre el archivo `index.html` en tu navegador:
   ```bash
   # Windows
   start index.html
   
   # macOS
   open index.html
   
   # Linux
   xdg-open index.html
   ```

### Personalizar Contenido
- **Información de contacto**: Actualiza el número de WhatsApp en los enlaces
- **Imágenes**: Reemplaza las imágenes en la carpeta `/images`
- **Textos y descripciones**: Edita el contenido directamente en `index.html`
- **Datos de servicios**: Modifica el objeto `servicesData` en `script.js`

## ✨ Funcionalidades JavaScript

### Scroll Optimizado
Detecta el scroll para agregar efectos visuales al navbar y activar animaciones de reveal.

### Modal de Servicios
Abre detalles de servicios con un modal interactivo que incluye:
- Título del servicio
- Descripción detallada
- Botón directo a WhatsApp con mensaje preformulado

### Loader de Bienvenida
Pantalla de carga personalizada con:
- Animación de fade del nombre
- Línea animada
- Duración configurable (1.2 segundos)
